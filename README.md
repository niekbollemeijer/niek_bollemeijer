# Dr. Niek Bollemeijer

My name is Niek Bollemeijer and I have recently defended my PhD thesis, titled `New spectral-timing techniques across the full X-ray band: revealing the dynamic corona in black hole X-ray binaries', at the University of Amsterdam, supervised by Dr. Phil Uttley. My research revolves around the hot X-ray corona that is observed in black hole X-ray binary systems, which I study in various ways that are described on this website.

This website is still work in progress.
